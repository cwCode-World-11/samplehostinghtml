Note : this is default website so you can just run this command "firebase deploy"

In this main level folder structure there is file called "firebase.json"

There is no site("site":"samplehostinghtml") property or object.

but,if we deploy this website it will deployed in "samplehostinghtml.web.app" by default

Why this is default?

In project folder there is file called ".firebaserc" . this will created after executing this command "firebase init"

In that file it specified the site("site":"samplehostinghtml"). it means this is default.

Otherwise we need to include "site":"samplehostinghtml" property in "firebase.json" file.
