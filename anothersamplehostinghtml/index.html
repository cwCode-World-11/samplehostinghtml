General intruction for deploying site in firebase.

  1. npm install -g firebase-tools (if your already installed in your local computer? just skip this)
  2. firebase login (if your already installed in your local computer? just skip this)
  3. firebase init
  4. firebase deploy

You can also deploy many site in same project in firebase

intruction for deploying another site in firebase.

  1. npm install -g firebase-tools (if your already installed in your local computer? just skip this)
  2. firebase login (if your already installed in your local computer? just skip this)
  3. Create a new file called "firebase.json"
    Add this code
    { "hosting":{ "site": "anothersamplehostinghtml", "public": "/", "ignore": [ "firebase.json", "**/.*", "**/node_modules/**" ] } }
  4. firebase deploy --only hosting:anothersamplehostinghtml
